AccessibilityWeb - accessibility from the user perspective!

AccessibilityWeb is a site about web accessibility from the user point of view. It provides explanations and examples of many of the things that disabled users and users of assistive technology might find confusing when using the web. It is especially handy for screen reader users who want to understand what their screen reader is reading out. Topics include the building blocks of websites, focus order, images and videos, forms, how designers try and make sites more accessible and how users can adjust their own settings. AccessibilityWeb does not discuss how to use assistive technology or how to make sites more accessible.
